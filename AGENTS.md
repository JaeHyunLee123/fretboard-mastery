# 기타/베이스 지판 암기 웹 앱 개발 계획 (Fretboard Master)

## 1. 프로젝트 개요 (Overview)

기타와 베이스 연주자들이 마이크나 오디오 인터페이스를 통해 실제 악기로 지판을 암기하고 연습할 수 있는 웹 기반 애플리케이션입니다.
현재 버전은 **백엔드 없이 프론트엔드 중심으로만 동작하는 단일 페이지(Single Page) 구조**로 기획되었습니다. Next.js 기반으로 클라이언트에서 모든 오디오 처리와 연습 로직을 수행하며 구글 AdSense를 연동하여 수익 창출을 도모합니다.

## 2. 디자인 및 UI 전략 (Stitch MCP 적용)

Stitch MCP에 정의된 `Fretboard Trainer Web App` 디자인 테마("The Digital Luthier")를 앱 전체에 적용합니다.

- **어두운 테마(Dark Mode)**: 눈의 피로를 덜어주는 어두운 배경(Obsidian)과 높은 대비의 포인트 컬러(Green/Red) 사용.
- **단일 화면 구성**: 모든 기능을 하나의 페이지에서 직관적으로 사용할 수 있도록 레이아웃을 구성합니다.

## 3. 주요 기능 (Core Features)

하나의 화면 내에서 아래 4가지 핵심 기능을 모두 조작할 수 있습니다.

1. **튜너 (Tuner)**
   - 마이크/오디오 입력을 받아 현재 연주하는 음의 피치와 주파수(Hz)를 실시간으로 보여줍니다.
2. **단일 노트 연습 (Single Note Practice)**
   - 앱이 랜덤으로 대상 음을 노출하고, 사용자가 악기로 해당 음을 정확히 연주했는지 판별합니다.
   - **표시 모드 토글**: 대상 음을 **음정 알파벳(C, D#, E 등)**, **타브 악보(TAB)**, **오선보(Staff)** 세 가지 방식으로 보여주며, 각각을 개별적으로 켜거나 끌 수(on/off) 있습니다.
3. **악기 전환 (Guitar & Bass Toggle)**
   - 스위치를 통해 '기타'와 '베이스' 모드를 전환합니다. 선택된 악기에 따라 연습하는 음역대와 튜닝 기준이 변경됩니다.
4. **광고 (Advertisements)**
   - Google AdSense 광고를 노출하기 위한 지정된 UI 지면(배너 슬롯 등)을 화면 내에 배치합니다.

## 4. 기술 스택 (Tech Stack)

- **Frontend**: Next.js (App Router), React, TypeScript, Tailwind CSS (Stitch 디자인 토큰 적용)
- **Audio**: Web Audio API, `pitchy` (피치 감지)
- **Score Rendering**: `vexflow` (오선보 및 타브 악보 렌더링)
- **Hosting**: Vercel (정적 웹 호스팅)
- **(Backend/DB 제외)**: 초기 버전에서는 사용자 인증 및 DB 저장을 제외하고 브라우저 상태(상태 관리 도구 활용)만 사용합니다.
